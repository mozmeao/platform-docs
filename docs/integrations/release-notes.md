# Release Notes

!!! springfield "Springfield Only"
    Release Notes are specific to firefox.com

Release note content is pulled in from an external data source and rendered using templates in the springfield repository.

## Templates

Release note templates live here: <https://github.com/mozmeao/springfield/tree/main/springfield/firefox/templates/firefox/releases>

## URLs

- Firefox release: <https://www.firefox.com/en-US/firefox/99.0.1/releasenotes/>
- Firefox Developer and Beta: <https://www.firefox.com/en-US/firefox/100.0beta/releasenotes/>
- Firefox Nightly: <https://www.firefox.com/en-US/firefox/101.0a1/releasenotes/>
- Firefox Android: <https://www.firefox.com/en-US/firefox/android/99.0/releasenotes/>
- Firefox iOS: <https://www.firefox.com/en-US/firefox/ios/99.0/releasenotes/>
